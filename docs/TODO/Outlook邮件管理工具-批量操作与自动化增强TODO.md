## 风险与依赖

### 技术风险
1. **验证码识别准确率**：依赖邮件格式，可能存在误判
   - 缓解措施：提供保底规则 + 用户可手动查看邮件
2. **浏览器兼容性**：Clipboard API 在旧浏览器不支持
   - 缓解措施：提供 execCommand 降级方案
3. **轮询性能影响**：高频轮询可能影响页面性能
   - 缓解措施：设置合理的默认值 + 使用异步请求

### 依赖项
- 依赖现有的邮件获取 API（Graph API / IMAP）
- 依赖现有的批量操作功能
- 依赖浏览器 Clipboard API
