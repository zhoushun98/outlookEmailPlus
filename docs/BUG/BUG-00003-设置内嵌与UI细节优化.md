# BUG-00003 — 设置内嵌与 UI 细节优化

> 关联 PRD: PRD-00004 UI全局美化重设计
> 创建日期: 2026-03-01
> 状态: 修复中

## 问题清单

### BUG-016: 设置页面应内嵌显示而非弹窗 (MEDIUM)
- **现象**: 点击侧边栏"系统设置"后，设置表单以弹窗打开，页面本身显示空白
- **根因**: 1) settings modal 在 modals.html 中，与 #page-settings 有重复 ID；2) 旧服务器未停止导致旧模板被缓存
- **修复**: 
  - 将设置表单从 modals.html 移入 index.html 的 `#page-settings` 
  - 清除 modals.html 中的重复表单（仅保留跳转提示）
  - `navigate('settings')` 改为调用 `loadSettings()` 而非 `showSettingsModal()`
- **状态**: ✅ 已修复

### BUG-017: 临时邮箱卡片缺少验证码提取按钮 (MEDIUM)
- **现象**: 正式账号卡片有 "🔑 验证码" 按钮可一键提取，但临时邮箱卡片没有
- **期望**: 临时邮箱卡片也应有 "🔑 验证码" 按钮，调用 `copyVerificationInfo()` 提取验证码
- **修复**: 在 `renderTempEmailList()` 的卡片 HTML 中添加验证码按钮
- **状态**: ✅ 已修复

### BUG-018: 导入账号弹窗 textarea 太小太丑 (LOW)
- **现象**: "导入邮箱账号"弹窗中的文本域高度仅 100px，显得很小且不协调
- **修复**: `.form-textarea` 最小高度增至 180px，添加完整样式（字体/边距/焦点态）
- **状态**: ✅ 已修复

### BUG-019: 账号头像颜色全部相同太单调 (LOW)
- **现象**: 所有账号卡片的头像圆圈都是琥珀金→砖红渐变色，视觉单调
- **修复**: 使用 8 组渐变色数组按索引分配（砖红、翠绿、海蓝、琥珀、紫罗兰、朱红、青绿、赭石）
- **状态**: ✅ 已修复
