## 一、架构概览

布局系统采用前端纯 JavaScript 实现，无后端依赖。

```
static/js/
├── layout-manager.js    # 核心控制器（~1400 行）
├── state-manager.js     # 状态持久化（~320 行）
└── layout-bootstrap.js  # 初始化入口（~66 行）

static/css/
└── layout.css           # 布局样式（~231 行）
```

### 数据流

```
用户操作 → LayoutManager (事件处理)
                ↓
         CSS 变量更新 → Grid 布局实时响应
                ↓
         StateManager → localStorage 持久化
```

## 二、核心类

### LayoutManager

负责拖拽调整、折叠/展开、窗口适配、键盘支持。

**关键方法：**

| 方法 | 说明 |
|------|------|
| `init()` | 初始化 DOM 缓存、事件绑定、状态加载 |
| `updatePanelWidth(panel, width)` | 更新面板宽度（参数为 DOM 元素和数值） |
| `collapsePanel(panelType, auto)` | 折叠面板（auto=true 表示窗口自适应触发） |
| `expandPanel(panelType, auto)` | 展开面板 |
| `togglePanel(panelType)` | 切换折叠/展开 |
| `resetLayout()` | 恢复默认布局 |
| `handleWindowResize()` | 窗口尺寸变化处理 |
| `onResizerKeydown(e)` | 键盘调整宽度 |
| `startResize(e)` / `resize(e)` / `stopResize()` | 鼠标拖拽三阶段 |

### StateManager

负责 localStorage 读写、数据校验、版本迁移。

**关键方法：**

| 方法 | 说明 |
|------|------|
| `save(userId, state)` | 保存状态 |
| `load(userId)` | 加载并迁移状态 |
| `clear(userId)` | 清除状态 |
| `validate(state)` | 校验数据结构 |
| `migrate(state)` | 版本迁移 |

**存储格式（v1.1）：**
```json
{
  "version": "1.1",
  "userId": "guest",
  "timestamp": 1709000000000,
  "panels": {
    "groups":   { "width": "200px", "collapsed": false },
    "accounts": { "width": "260px", "collapsed": false },
    "emails":   { "width": "380px", "collapsed": false }
  }
}
```

**存储键名：** `outlook_layout_state_{userId}`

## 三、CSS 变量系统

布局通过 CSS 变量控制，定义在 `:root` 中：

```css
/* 面板保存宽度 */
--group-panel-width: 200px;
--account-panel-width: 260px;
--email-list-panel-width: 380px;

/* 面板布局生效宽度（折叠时为 0px） */
--group-panel-layout-width: 200px;
--account-panel-layout-width: 260px;
--email-list-panel-layout-width: 380px;

/* 约束 */
--group-panel-min-width: 150px;
--group-panel-max-width: 400px;
/* ... 其他面板类似 */

/* 动画 */
--panel-transition-duration: 200ms;
--resizer-width: 4px;
```

**修改默认宽度：** 修改 `layout.css` 中的 `:root` CSS 变量和 `layout-manager.js` 中的 `DEFAULT_WIDTHS` 常量。

## 五、扩展指南

### 添加新面板

1. **HTML**: 在 `templates/index.html` 的 `.main-container` 中添加面板元素
2. **CSS**: 在 `layout.css` 中添加对应的 CSS 变量
3. **JS**: 在 `layout-manager.js` 的常量映射（`PANEL_WIDTH_VAR_MAP` 等）中注册

### 修改宽度约束

在 `layout.css` 中修改对应的 `--*-min-width` 和 `--*-max-width` 变量，同时确保 `layout-manager.js` 的 `getMinWidth()` / `getMaxWidth()` 方法同步更新。

### 修改断点

在 `layout-manager.js` 的 `handleWindowResize()` 方法中修改断点值（当前为 1200/900/700px）。
