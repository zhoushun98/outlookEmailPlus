# 布局系统使用指南

## 文档信息

- **文档编号**: USER-00001
- **创建日期**: 2026-03-01
- **版本**: v1.0
- **关联功能**: 可调整布局系统

## 一、功能概述

Outlook 邮件管理工具提供了四栏可调整布局系统，包括：

| 面板 | 默认宽度 | 可折叠 | 说明 |
|------|----------|--------|------|
| 分组面板 | 200px | ✅ | 邮箱分组列表 |
| 账号面板 | 260px | ✅ | 邮箱账号列表 |
| 邮件列表面板 | 380px | ✅ | 邮件列表 |
| 邮件详情面板 | 自适应 | ❌ | 邮件内容详情 |

## 二、操作指南

### 2.1 调整面板宽度

**鼠标拖拽：**
1. 将鼠标移到两个面板之间的分隔条上（光标会变为 `↔` 样式）
2. 按住鼠标左键拖动即可调整宽度
3. 松开鼠标后，宽度自动保存

**宽度限制：**
- 分组面板：150px ~ 400px
- 账号面板：180px ~ 500px
- 邮件列表面板：280px ~ 600px
- 邮件详情面板始终保持 ≥ 400px

### 2.2 折叠/展开面板

**折叠面板：**
- 点击面板标题栏右侧的 `←` 按钮

**展开面板：**
- 点击折叠后出现的竖向指示条（显示面板名称）
- 或点击 `→` 按钮

**注意事项：**
- 邮件详情面板不可折叠
- 至少需要保持一个面板展开

### 2.3 恢复默认布局

1. 点击顶部导航栏的 `↻ 恢复默认布局` 按钮
2. 在确认对话框中点击「确定」
3. 所有面板将恢复默认宽度并展开

### 2.4 键盘操作

| 操作 | 快捷键 | 说明 |
|------|--------|------|
| 聚焦分隔条 | `Tab` | 按 Tab 键可以聚焦到分隔条 |
| 减小宽度 | `←` | 每次减小 10px |
| 增大宽度 | `→` | 每次增大 10px |
| 快速减小 | `Shift + ←` | 每次减小 50px |
| 快速增大 | `Shift + →` | 每次增大 50px |
| 折叠/展开 | `Enter` / `Space` | 聚焦折叠按钮后按下 |
| 关闭确认框 | `Esc` | 关闭恢复默认布局的确认对话框 |

## 三、自动适配

系统会根据浏览器窗口大小自动调整布局：

| 窗口宽度 | 自动行为 |
|----------|----------|
| < 1200px | 自动折叠分组面板 |
| < 900px | 自动折叠账号面板 |
| < 700px | 自动折叠邮件列表面板 |

**自动恢复：** 当窗口放大回原来的尺寸时，被自动折叠的面板会自动恢复（手动折叠的面板不会自动恢复）。

## 四、状态保存

- 布局状态（面板宽度、折叠状态）自动保存在浏览器本地存储中
- 刷新页面后布局会自动恢复到上次的状态
- 不同用户的布局独立存储，互不影响
- 如需清除保存的布局，使用「恢复默认布局」功能

## 五、常见问题

**Q: 拖拽分隔条没有反应？**
A: 请确保鼠标精确对准分隔条（宽度为 4px），光标变为 `↔` 时再拖动。

**Q: 恢复默认布局后之前的设置还能找回吗？**
A: 不能，恢复默认布局会清除保存的状态。

**Q: 为什么某个面板无法再缩小了？**
A: 每个面板都有最小宽度限制，防止内容无法显示。

**Q: 换了浏览器后布局设置丢失了？**
A: 布局设置保存在浏览器的本地存储中，不同浏览器之间不共享。

**Q: 无痕/隐私模式下布局能保存吗？**
A: 部分浏览器在无痕模式下会限制本地存储，布局可能无法持久保存，但在当次会话中仍可正常使用。
